# foodiePalApp created by Edet, Ebenezer

The foodiePalApp was designed with the intention to create a culinary experience for users with the below features:

1. A Login page
2. Add new recipes
3. Setup a meal planner
4. Add a blog post
5. Call / Email the chef
6. An About me page
